Frequently Asked Questions¶

Does it support JOINs?¶

No, because MongoDB doesn’t, but have look at Embedded Objects and Storing Python Lists and Dicts.

Does it support transcations?¶

No, because MongoDB doesn’t.

How can I use Capped Collections?¶

See Collection Options.

Does it support query debugging?¶

Yep! Set DEBUG to True in your settings.py and configure the django.db.backends logger, for example like this:

LOGGING = {
    'version' : 1,
    'formatters' : {'simple' : {'format': '%(levelname)s %(message)s'}},
    'handlers' : {
        'console' : {
            'level' : 'DEBUG',
            'class' : 'logging.StreamHandler',
            'formatter' : 'simple'
        }
    },
    'loggers' : {
        'django.db.backends' : {
            'level' : 'DEBUG',
            'handlers' : ['console']
        }
    }
}
